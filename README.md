# coc-file-utils

The best way to create, copy, move, rename and delete files and folders (multiple files)

## Install

`:CocInstall coc-file-utils`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-file-utils-keymap)`

## Lists

`:CocList demo_list`
