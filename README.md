# coc-file-utils

The best way to create, copy, move, rename and delete file and folder

## TODO

## Install

`:CocInstall coc-file-utils`

## Lists

`:CocList dirs`

## commands

`:CocCommand coc-file-utils.delete` delete file

`:CocCommand coc-file-utils.rename` rename file
